Via ntfy.sh (mobile + desktop)

  1. Installez ntfy.sh sur votre telephone
  2. Abonnez-vous au topic : cgp-monitor-cmf

Extension Folk (import 1 clic)

Installez l'extension Folk pour Chrome. Ouvrez la fiche d'un CGP sur ce site, cliquez sur l'icone Folk dans la barre d'outils : l'extension detecte automatiquement le nom du cabinet, les dirigeants, email, telephone, site web et adresse (markup Schema.org incorpore) et les importe dans votre CRM Folk.

Astuce : le bouton "Folk" sur chaque carte permet de marquer manuellement les CGP deja importes dans votre CRM.

Synchronisation multi-appareils

Synchronisez vos statuts (contacte, en cours, refus, Folk) entre votre iPhone et vos differents PC via un GitHub Gist prive.

Configuration

  1. Creez un Gist GitHub (prive, nom de fichier : cgp-monitor-state.json, contenu : {}).
  2. Copiez l'ID du Gist (dans l'URL, ex: abc123def456).
  3. Generez un token GitHub Personal Access Token avec le scope gist.
  4. Collez les deux dans les champs ci-dessous puis Enregistrer. Repetez sur chaque appareil.

Les identifiants restent sur votre appareil (localStorage). Ils ne sont jamais envoyes ailleurs que vers api.github.com.
